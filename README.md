# CUA Agent Playwright Demo

This project is a simple Computer Use Agent demo.

The goal is to show how an agent can control a browser through code.

## Requirements

- Use Python and Playwright.
- Open a browser automatically.
- Open a local chat webpage.
- Type the first question: "What is an Agent?"
- Click the send button.
- Wait for the answer.
- Type the second question: "What is the difference between an Agent and a chatbot?"
- Print observations in the terminal.
- Keep the code beginner-friendly.
